# Section_01-01_Deck

This is a basic Rust project initialized using `cargo new`. It serves as a foundational setup for learning Rust.

## Prerequisites

To run this project, you need to have the Rust toolchain installed.

- Install Rust (This installs `rustc`, `cargo`, and `rustup`)

## Getting Started

Navigate to the project directory:

```sh
cd Section_01-01_Deck
```

### Building

To compile the project without running it:

```sh
cargo build
```

### Running

To compile and run the project in one step:

```sh
cargo run
```

Expected output:

```text
Hello, world!
```
