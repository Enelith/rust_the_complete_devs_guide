## Local Rust Install

To install Rust locally on your computer, follow the directions on this page:

```
https://www.rust-lang.org/tools/install
```

After installation, restart your terminal then try running `cargo --version`. 

If you see a version print up then you're good to go! 

If you get an error then a little troubleshooting is in order.

## Cargo

*Cargo is used to create, compile, run and manage projects.*

| In order to... | ... Use the following command |
